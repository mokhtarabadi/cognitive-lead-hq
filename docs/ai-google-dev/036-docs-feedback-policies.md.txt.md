# Feedback Policies

## Inline Preference Voting

Google AI Studio occasionally shows side-by-side response comparisons. Choose your preferred response to help improve model outputs.

### Purpose

Feedback improves Google products, services, and ML technologies per the [Gemini API Additional Terms of Service](https://ai.google.dev/gemini-api/terms) and [Privacy Policy](https://policies.google.com/privacy).

### Data Collected

- **Prompts & Responses** — All prompts, responses (including uploaded content), and both response options shown
- **Your Vote** — Which response you preferred
- **Usage Details** — Model used and technical/operational usage data

> [!warning]
> Do not submit feedback on conversations containing sensitive, confidential, or personal information. Data is disconnected from your Google Account, API key, and Cloud project before review.

### Opting Out

Skip Inline Preference Voting when prompted.
