# Pricing

Start free with generous limits, then scale with prepaid or pay-as-you-go pricing.

## Pricing Tiers

### Free

- Limited access to certain models
- Free input & output tokens
- Google AI Studio access
- Content used to improve products

[Get started for Free](https://aistudio.google.com)

### Paid

- Higher rate limits
- Context caching
- Batch API (50% cost reduction)
- Access to advanced models
- Content **not** used to improve products

[Upgrade to Paid](https://aistudio.google.com/api-keys)

### Enterprise

Powered by [Gemini Enterprise Agent Platform](https://cloud.google.com/gemini-enterprise-agent-platform):

- All Paid features
- Dedicated support channels
- Advanced security & compliance
- Provisioned throughput
- Volume-based discounts

[Contact Sales](https://cloud.google.com/contact)
